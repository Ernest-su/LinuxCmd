dmesg
===

显示Linux系统启动信息

## 补充说明

**dmesg命令** 被用于检查和控制内核的环形缓冲区。kernel会将开机信息存储在ring buffer中。您若是开机时来不及查看信息，可利用dmesg来查看。开机信息保存在`/var/log/dmesg`文件里。

###  语法 

```shell
dmesg(选项)
```

###  选项 

```shell
-c：显示信息后，清除ring buffer中的内容；
-s<缓冲区大小>：预设置为8196，刚好等于ring buffer的大小；
-n：设置记录信息的层级。
```
